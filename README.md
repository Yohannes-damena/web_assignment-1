# Web Programming Assignment 1

This repository contains three HTML projects developed as part of a Web Programming assignment. Each file demonstrates different aspects of HTML5, ranging from semantic structure and interactive elements to page layout and multimedia integration.

## Files Overview

### 1. [The Ultimate HTML5 Tag Reference](all-tags.html) (`all-tags.html`)
A comprehensive, interactive guide designed for beginners to master HTML tags and attributes.
- **Key Features:**
  - Interactive `contenteditable` section.
  - Deep dive into semantic structure (`<article>`, `<aside>`, `<nav>`).
  - Multimedia demonstrations including `<picture>`, `<map>`, `<audio>`, and `<video>`.
  - Advanced form elements (`<datalist>`, `<dialog>`, `<output>`).
  - Mastery checklist using HTML tables and checkboxes.

### 2. [Department of Computer Science](index.html) (`index.html`)
A functional department website mockup providing information about academic programs and courses.
- **Key Features:**
  - Navigation menu for quick section access.
  - Program duration and enrollment capacity visualized with `<progress>` bars.
  - Multimedia integration with a local video, audio podcast, and an embedded YouTube tour.
  - A structured contact form for prospective students.

### 3. [Puppy Lovers Page](puppy_lovers/index.html) (`puppy_lovers/index.html`)
A visually centered landing page dedicated to puppy enthusiasts.
- **Key Features:**
  - Clean, centered layout using HTML tables for alignment.
  - Responsive banner and a gallery of high-quality puppy images.
  - Custom styling for headers and footers to create a cohesive theme.

## Technologies Used
- HTML5 (Semantic elements, Multimedia, Forms)
- CSS3 (Embedded styles for layout and presentation)
